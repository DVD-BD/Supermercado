Abrimos la documentación de github pages para entender el procedimiento para desplegar una página web
Creamos una Carpeta Supermercado y cargamos el documento index con el código html
Instalamos git en la computadora mediante un ejecutable 
A través del cmd subimos la carpeta a un repositorio de github pages
Abrimos el apartado Deployments y entramos a la url generada para el sitio web.
