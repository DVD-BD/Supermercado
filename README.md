## Procedimiento para publicar la página web

1. Abrimos la documentación de GitHub Pages para entender el procedimiento para desplegar una página web.

2. Creamos una carpeta llamada **Supermercado** y cargamos el documento **index.html** con el código HTML de la página.

3. Instalamos **Git** en la computadora mediante un archivo ejecutable.

4. A través del **CMD** subimos la carpeta al repositorio de GitHub Pages utilizando los comandos de Git.

5. Abrimos el apartado **Deployments** en el repositorio y accedemos a la **URL generada** para visualizar el sitio web publicado.
